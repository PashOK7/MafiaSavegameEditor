# Mafia Save Editor v0.1.0

Initial public release of the project.

## Highlights

- Windows GUI editor for `mafiaXXX.YYY` save files.
- Tabs: `Main`, `Actors`, `Cars`, `Garage`, `Mission/Script`.
- `G_Stream`-compatible parse/edit/rebuild pipeline.
- Garage editor with car name decoding from `Mafia/tables/carindex.def` (fallback `carcyclopedia.def`).

## Build

```powershell
g++ -std=c++17 -O2 -Wall -Wextra mafia_save.cpp mafia_editor_gui.cpp -o bin/gui/mafia_editor_gui.exe -mwindows -lcomdlg32 -lcomctl32
```
